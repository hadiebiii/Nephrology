# Documentation Writer Skill

Use this skill when the user asks to write or improve documentation.

## Documentation targets

- `README.md`
- `AGENTS.md`
- `.github/copilot-instructions.md`
- `.github/instructions/*.md`
- Deployment notes
- Usage guides
- Local development instructions

## Style

- Clear
- Practical
- Step-by-step
- Beginner-friendly
- Persian or English based on user request
- Include commands when useful
- Avoid unnecessary theory

## Recommended README structure

- Project title
- Purpose
- Live URL
- Folder structure
- Local preview
- Editing guide
- Validation
- Deploy workflow
- GitHub Pages status
- Agent, Copilot, and Codex notes

## Safety

- Do not expose tokens.
- Do not include private credentials.
- Do not change workflow files unless explicitly requested.
- Do not push unless explicitly requested.
